Takathon
========

A language focused on QA and unit testing.

Features
--------

* Standarized test structure
* Dedicated Python like syntax
* Test domains
* Compatibility with Python
* Support for BDD workflow

Install
-------

.. code-block:: bash

   pip install -e git://github.com/piwonskp/takathon.git@0.1.0#egg=takathon
